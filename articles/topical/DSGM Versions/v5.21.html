DS Game Maker v5.21

DS Game Maker v5.21 was released on August 2nd, 2022 at 05:14. It was created by DigitalDesignDude using an old version of the source code of DS Game Maker. The installation instructions can be found here.


New Features:

• Ability to re-arrange side panel assets.
• Fixed inability to Copy and Paste Actions
• Room Preview window now tiles any background size
• Fixed inability to import 16x8 and 32x8 sized sprites.
• Added new resource links to the "Help" tab.
