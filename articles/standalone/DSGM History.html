DS Game Maker History

Before DS Game Maker, there was a game engine made. It was known as XtenDS. This engine, while not super feature rich, allowed you to create homebrew for the Nintendo DS. The creator of this software is known as James Garner. After a few versions of XtenDS were released, James found that he wasn't reaching as many people as YoYo's popular Game Maker was. This led James to rename XtenDS to DS Game Maker. After he renamed it, he released the first version of DS Game Maker. DS Game Maker started to pick up steam. DS Game Maker ended up being a popular choice for getting into NDS homebrew or game dev in general. It is not known when the forums went up. But we speculate it was around the time DS Game Maker v1.0 was released. The forums lasted until mid-2016 when they was taken down. Many pages are still available to view via the internet archive.

After the forums when down, there was a video uploaded by SandySage showing how to install DS Game Maker now that the forums and site were offline. This is an important piece of the puzzle which we will get back too. After this video was released, DSGM Online!, the first DS Game Maker Discord server, was created. It was created on June 23rd, 2017. dsgm.boards.net was also created around this time. On May 22nd, 2020, James and Ruffsta finished work on the dsgamemaker.online site. This site had games, resources, and a forum for DS Game Maker. Around a month later, on June 15th of 2020, the DS Game Maker Virtual Machine was released by James. This helped spark the community quite a bit. But this didn't last long. Because once it got to 2022, the forums were basically dead.

Remember that video from SandySage? At this time, DigitalDesignDude made a video going over how to install DS Game Maker on Windows 10, which was inspired by SandySage's video. And then, at some point in January, Kenyon Bowers found the Virtual Machine and used it to create his own DS game. When he saw that this software needed a bigger and better community, he created the DS Game Maker Discord server. This server was created on February 17th, 2022. This server steadily grew over the next few months. In June of 2022, DigitalDesignDude released a written installation tutorial for DS Game Maker. On July 25th, 2022, DigitalDesignDude released the DS Game Maker Resource Site. Later, he also released both DS Game Maker versions v5.21A and v5.21B. When Kenyon saw how big the community was starting to get, he decided to make the DS Game Maker Subreddit. About a month later, Kenyon created a Twitter account for DS Game Maker. This account was deleted on November 1st, 2022, due to him not liking what he was seeing on Twitter.
