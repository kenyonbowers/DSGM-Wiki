The Revival

After the original forums went down, there were a few attempts to revive the DS Game maker community. We will go over some of those attempts. However, not every attempt is on here due to not even knowing they happened.

One attempt was back in June of 2016 (June 16th, 2016, to be exact). A person by the name of SandySage uploaded a video showing how to install DS Game Maker. This video by itself did not revive the community. The next attempt to bring the community back was about a years later in 2017. Both dsgm.boards.net and DSGM Online! were created. These two were interconnected having some of the same members, resources, etc.

This then brings us to the next attempt. In 2020, James and Ruffsta released a website for DS Game Maker. This website is still around and is known as dsgamemaker.online. A little less than a month later, James released the DS Game Maker VM. This allowed people to download it and run it through VirtualBox without needing to figure out how to get DS Game Maker/old version of devkitPro. This helped spark the community a bit and the built-in forums were moderately active. After the forums had died - end of 2021 - Kenyon found and picked up the DS Game Maker VM from dsgamemaker.online. Within two weeks he created his first DS game with the software. Knowing the DS Game Maker community needed reviving, he created the DS Game Maker Discord server. This server ended up growing tremendously. Only a short time of about a few months, DigitalDesignDude created the DSGM Resource Site. Once Kenyon realized that the DS Game Maker server was growing at a fast rate, he created the DS Game Maker Subreddit. A month later, he created a DSGM News Twitter account. This was deleted only a few days later due to him not liking the content that was on Twitter.
