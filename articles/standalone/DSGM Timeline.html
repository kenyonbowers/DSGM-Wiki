Era Two | Fan DSGM:

This era includes, DS Game Maker 5.20/5.21 and fan communities.


June 23rd, 2017:

DSGM Online!, DS Game Maker's first Discord server, was created. The creator of this server was Buck_7.


November 5th, 2017:

dsgm.boards.net, a fan forum, went online.


May 22nd, 2020

James and Ruffsta made dsgamemaker.online (originally known as https://dsgamemaker.jada.io/).


June 15th, 2020

DS Game Maker was rereleased in the DS Game Maker VM. This was done by James Garner .


February 17th, 2022:

DS Game Maker's biggest Discord called DS Game Maker, was created.


July 25th, 2022:

DSGM Resource Site was created. This website has tutorials, documentation, and almost any information you would need when making a DS Game Maker game.


August 2nd, 2022 at 05:14:

DigitalDesignDude released DS Game Maker v5.21A.


August 8th, 2022 at 06:41:

DigitalDesignDude released DS Game Maker v5.21B.


August 13th, 2022 at 08:45:

Kenyon Bowers released DS Game Maker v5.21B inside the DSGM v5.21 VM.


September 17th, 2022:

r/DSGameMaker was created. It was created to compliment the DS Game Maker Discord server. This was created in hopes of bringing the Reddit community into DS Game Maker.
